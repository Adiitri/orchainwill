# orchainwill

## Project Description

OrchainWill is a decentralized digital will system built using the Stellar Soroban smart contract platform.  
It allows users to securely store a final message, instructions, or asset reference on the blockchain and designate a beneficiary who can access it later.

Traditional wills rely on centralized institutions such as lawyers or banks. OrchainWill leverages blockchain technology to provide a transparent, secure, and tamper-proof way of storing testamentary information.

## What it does

OrchainWill enables a user to create a digital will on the Stellar blockchain. The user specifies:

- The owner of the will
- A beneficiary
- A stored message or instruction

The beneficiary can later retrieve the stored message using the smart contract.

This ensures the will remains immutable and accessible without relying on a centralized authority.

## Features

- Decentralized digital will storage
- Immutable blockchain record
- Beneficiary-based access control
- Simple and secure Soroban smart contract
- Transparent execution on the Stellar network
- Minimal gas and efficient contract logic
